## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

Рабочее тело - вспомогательное вещество, используемое для работы тепловой машины (двигателя, холодильной установки, теплового насоса). В большинстве случаев рабочее тело является газообразным веществом.

Рабочие тела делят на идеальные и реальные газы. Идеальный газ - воображаемый газ, в котором отсутствуют силы межмолекулярного взаимодействия, а молекулы имеют пренебрежимо малый объем. Реальный газ состоит из молекул, объемом которых нельзя пренебречь, и между ними существуют силы взаимодействия.

### Основные параметры состояния:
1. **Абсолютное давление (P)** [Па] - сила, действующая на единицу поверхности
   Pабс = Рман + Рбар (если давление больше атмосферного)
   Pабс = Рбар – Рвак (если давление меньше атмосферного)

2. **Удельный объем (v)** [м³/кг] - объем, занимаемый единицей массы
   v = V/m

3. **Температура (T)** [К] - мера средней кинетической энергии молекул
   T(K) = t(°C) + 273.15

### Единицы измерения давления:
1 Па = 1 Н/м²
1 бар = 10⁵ Па
1 ат = 1 кгс/см² = 98066.5 Па ≈ 10⁵ Па
1 мм рт.ст. = 133.322 Па
1 мм вод.ст. = 9.81 Па

### Уравнение состояния идеального газа (Клапейрона-Менделеева):
PV = mRT
или для 1 кг: Pv = RT

Универсальная газовая постоянная: R = 8.314 Дж/(моль·К)
Удельная газовая постоянная: Rуд = R/μ, где μ - молярная масса [кг/моль]

### Нормальные условия:
Pн = 101325 Па (760 мм рт.ст.)
Tн = 273.15 К (0°С)

### Физические константы [citation:9]:
- Постоянная Больцмана: k = 1.38×10⁻²³ Дж/К
- Универсальная газовая постоянная: R = 8.31 Дж/(моль·К)
- Число Авогадро: Nₐ = 6.02×10²³ моль⁻¹

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 1.1
Определить массу кислорода в баллоне объемом 40 л при температуре 27°С и давлении 12 МПа.

**Дано:** V = 40 л = 0.04 м³, t = 27°C → T = 300 K, P = 12 МПа = 12·10⁶ Па, μ(O₂) = 32·10⁻³ кг/моль

**Решение:**
Из уравнения Клапейрона-Менделеева: PV = (m/μ)RT
m = (P V μ) / (R T) = (12·10⁶ · 0.04 · 32·10⁻³) / (8.31 · 300) = (15.36·10³) / (2493) ≈ 6.16 кг

**Ответ:** m = 6.16 кг

### Пример 1.2
Вакуумметр показывает разрежение 0.5 кгс/см². Барометрическое давление по ртутному барометру составляет 740 мм рт.ст. Определить абсолютное давление.

**Дано:** Рвак = 0.5 кгс/см² = 0.5·98066.5 ≈ 49033 Па, Рбар = 740 мм рт.ст. = 740·133.322 ≈ 98658 Па

**Решение:**
Рабс = Рбар - Рвак = 98658 - 49033 = 49625 Па ≈ 0.496 бар

**Ответ:** Рабс = 49625 Па

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

Газовая смесь - механическая смесь отдельных газов, не вступающих между собой в химические реакции.

### Способы задания состава смеси:

1. **Массовые доли**: gᵢ = mᵢ / m_см, где Σgᵢ = 1

2. **Объемные доли**: rᵢ = Vᵢ / V_см, где Σrᵢ = 1
   (Vᵢ - приведенный объем i-го газа)

3. **Мольные доли**: yᵢ = nᵢ / n_см, где Σyᵢ = 1

### Пересчет составов:
- Массовые доли в объемные: rᵢ = (gᵢ/μᵢ) / Σ(gᵢ/μᵢ)
- Объемные доли в массовые: gᵢ = (rᵢ·μᵢ) / Σ(rᵢ·μᵢ)

### Кажущаяся молярная масса смеси:
μ_см = 1 / Σ(gᵢ/μᵢ) = Σ(rᵢ·μᵢ)

### Газовая постоянная смеси:
R_см = R / μ_см = Σ(gᵢ·Rᵢ)

### Парциальное давление:
Закон Дальтона: P = Σpᵢ
pᵢ = rᵢ·P - парциальное давление i-го газа

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 2.1
Газовая смесь имеет состав по объему: СО₂ - 12%, О₂ - 8%, N₂ - 80%. Определить массовый состав, кажущуюся молярную массу и газовую постоянную смеси.

**Дано:** r(CO₂)=0.12, μ(CO₂)=44·10⁻³ кг/моль; r(O₂)=0.08, μ(O₂)=32·10⁻³ кг/моль; r(N₂)=0.80, μ(N₂)=28·10⁻³ кг/моль

**Решение:**
1. Кажущаяся молярная масса:
μ_см = Σ(rᵢ·μᵢ) = 0.12·44·10⁻³ + 0.08·32·10⁻³ + 0.80·28·10⁻³ = 0.00528 + 0.00256 + 0.0224 = 0.03024 кг/моль = 30.24·10⁻³ кг/моль

2. Массовые доли:
g(CO₂) = r(CO₂)·μ(CO₂)/μ_см = 0.12·44/30.24 = 5.28/30.24 ≈ 0.175
g(O₂) = 0.08·32/30.24 = 2.56/30.24 ≈ 0.085
g(N₂) = 0.80·28/30.24 = 22.4/30.24 ≈ 0.740

Проверка: 0.175 + 0.085 + 0.740 = 1.0

3. Газовая постоянная смеси:
R_см = R/μ_см = 8.314 / 0.03024 ≈ 275 Дж/(кг·К)

**Ответ:** g(CO₂)=0.175, g(O₂)=0.085, g(N₂)=0.740, μ_см=30.24·10⁻³ кг/моль, R_см=275 Дж/(кг·К)

### Пример 2.2
Определить парциальные давления газов в смеси из примера 2.1, если общее давление смеси P = 1 МПа.

**Решение:**
p(CO₂) = r(CO₂)·P = 0.12·1 = 0.12 МПа
p(O₂) = 0.08·1 = 0.08 МПа
p(N₂) = 0.80·1 = 0.80 МПа

**Ответ:** p(CO₂)=0.12 МПа, p(O₂)=0.08 МПа, p(N₂)=0.80 МПа

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

**Теплоемкость** - количество теплоты, необходимое для изменения температуры тела на 1 К.

### Виды теплоемкости [citation:6]:
1. **Удельная теплоемкость** (с) [Дж/(кг·К)] - отнесена к 1 кг вещества
2. **Молярная теплоемкость** (С) [Дж/(моль·К)] - отнесена к 1 молю вещества
3. **Объемная теплоемкость** (с') [Дж/(м³·К)] - отнесена к 1 м³ газа при нормальных условиях

### Зависимость от процесса:
- **Теплоемкость при постоянном объеме (cᵥ, Cᵥ)** - теплота расходуется только на изменение внутренней энергии
- **Теплоемкость при постоянном давлении (cₚ, Cₚ)** - теплота расходуется на изменение внутренней энергии и на совершение работы

### Соотношение Майера:
Cₚ - Cᵥ = R
cₚ - cᵥ = R (для 1 кг)

### Показатель адиабаты:
k = Cₚ / Cᵥ = cₚ / cᵥ

### Молекулярно-кинетическая теория теплоемкости [citation:3]:
Для идеального газа с числом степеней свободы i:
- Cᵥ = (i/2)·R
- Cₚ = ((i+2)/2)·R
- k = (i+2)/i

Число степеней свободы:
- Одноатомный газ (He, Ne, Ar): i = 3
- Двухатомный газ (O₂, N₂, воздух): i = 5
- Многоатомный газ (CO₂, H₂O): i = 6

### Зависимость теплоемкости от температуры:
При повышении температуры теплоемкость увеличивается. Используют:
- **Среднюю теплоемкость** в интервале температур
- **Истинную теплоемкость** при данной температуре

Аппроксимационные формулы:
c = a + bt + dt²

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 3.1
Определить среднюю массовую теплоемкость cₚ для воздуха в интервале температур от 200°С до 800°С.

**Дано:** t₁ = 200°C, t₂ = 800°C

**Решение:**
Используем табличные зависимости: cₚ = 0.995 + 0.000093·t [кДж/(кг·К)]

cₚ₁ = 0.995 + 0.000093·200 = 0.995 + 0.0186 = 1.0136 кДж/(кг·К)
cₚ₂ = 0.995 + 0.000093·800 = 0.995 + 0.0744 = 1.0694 кДж/(кг·К)

Средняя теплоемкость: cₚ_ср = (cₚ₁ + cₚ₂)/2 = (1.0136 + 1.0694)/2 = 1.0415 кДж/(кг·К)

**Ответ:** cₚ_ср = 1.0415 кДж/(кг·К)

### Пример 3.2
Определить показатель адиабаты для углекислого газа CO₂ (многоатомный газ).

**Решение:**
Для многоатомного газа i = 6
Cᵥ = (i/2)·R = 3R = 3·8.314 = 24.942 Дж/(моль·К)
Cₚ = ((i+2)/2)·R = 4R = 4·8.314 = 33.256 Дж/(моль·К)
k = Cₚ/Cᵥ = 33.256/24.942 ≈ 1.33

**Ответ:** k = 1.33

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### Формулировки первого закона термодинамики [citation:3]:
1. Энергия не исчезает и не создается, а переходит из одной формы в другую
2. Невозможно построить вечный двигатель первого рода
3. Изменение внутренней энергии системы равно сумме подведенной теплоты и совершенной работы

### Математическое выражение [citation:4]:

**Интегральная форма:**
Q = ΔU + L

где:
Q - подведенная теплота [Дж]
ΔU - изменение внутренней энергии [Дж]
L - работа, совершенная системой против внешних сил [Дж]

**Для 1 кг рабочего тела:**
q = Δu + l

**Дифференциальная форма:**
δq = du + δl
или
δq = du + P·dv

**Для потока (техническая работа):**
δq = dh + δl_техн

### Внутренняя энергия [citation:9]:
Функция состояния, включающая кинетическую энергию движения молекул и потенциальную энергию их взаимодействия.

Для идеального газа: u = f(T) только
du = cᵥ·dT
Δu = cᵥ·(T₂ - T₁)

### Энтальпия:
h = u + P·v [Дж/кг]
Функция состояния, широко используемая в теплотехнике.

Для идеального газа: h = f(T) только
dh = cₚ·dT
Δh = cₚ·(T₂ - T₁)

### Работа расширения [citation:4]:
δl = P·dv
l₁₂ = ∫₁² P·dv

В Pv-координатах работа численно равна площади под кривой процесса.

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 4.1
Газ совершает работу расширения 500 Дж, при этом его внутренняя энергия увеличивается на 200 Дж. Какое количество теплоты подведено к газу?

**Дано:** L = 500 Дж, ΔU = 200 Дж

**Решение:**
По первому закону: Q = ΔU + L = 200 + 500 = 700 Дж

**Ответ:** Q = 700 Дж

### Пример 4.2
1 кг воздуха нагревается от 20°С до 100°С при постоянном давлении. Определить количество подведенной теплоты, изменение внутренней энергии и совершенную работу. cₚ = 1.005 кДж/(кг·К), cᵥ = 0.718 кДж/(кг·К).

**Дано:** m = 1 кг, t₁ = 20°C → T₁ = 293 K, t₂ = 100°C → T₂ = 373 K, P = const

**Решение:**
1. Подведенная теплота:
qₚ = cₚ·(T₂ - T₁) = 1.005·(373 - 293) = 1.005·80 = 80.4 кДж/кг

2. Изменение внутренней энергии:
Δu = cᵥ·(T₂ - T₁) = 0.718·80 = 57.44 кДж/кг

3. Работа расширения:
l = qₚ - Δu = 80.4 - 57.44 = 22.96 кДж/кг
или l = R·(T₂ - T₁) = (cₚ - cᵥ)·(T₂ - T₁) = 0.287·80 = 22.96 кДж/кг

**Ответ:** q = 80.4 кДж/кг, Δu = 57.44 кДж/кг, l = 22.96 кДж/кг

# РАЗДЕЛ 5: ПРОЦЕССЫ ИЗМЕНЕНИЯ СОСТОЯНИЯ ИДЕАЛЬНЫХ ГАЗОВ

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### Классификация термодинамических процессов [citation:3]:
1. **Изохорный** (v = const)
2. **Изобарный** (P = const)
3. **Изотермический** (T = const)
4. **Адиабатный** (δq = 0)
5. **Политропный** (c = const)

### 5.1 ИЗОХОРНЫЙ ПРОЦЕСС (v = const)

**Уравнение процесса:** P/T = const или P₁/T₁ = P₂/T₂

**Соотношения параметров:**
P₂/P₁ = T₂/T₁

**Работа:** l = 0

**Изменение внутренней энергии:**
Δu = cᵥ·(T₂ - T₁)

**Теплота:**
q = Δu = cᵥ·(T₂ - T₁)

**Изменение энтропии:**
Δs = cᵥ·ln(T₂/T₁)

### 5.2 ИЗОБАРНЫЙ ПРОЦЕСС (P = const)

**Уравнение процесса:** v/T = const или v₁/T₁ = v₂/T₂

**Соотношения параметров:**
v₂/v₁ = T₂/T₁

**Работа:**
l = P·(v₂ - v₁) = R·(T₂ - T₁)

**Изменение внутренней энергии:**
Δu = cᵥ·(T₂ - T₁)

**Теплота:**
q = cₚ·(T₂ - T₁)

**Изменение энтальпии:**
Δh = q = cₚ·(T₂ - T₁)

**Изменение энтропии:**
Δs = cₚ·ln(T₂/T₁)

### 5.3 ИЗОТЕРМИЧЕСКИЙ ПРОЦЕСС (T = const)

**Уравнение процесса:** P·v = const или P₁/P₂ = v₂/v₁

**Соотношения параметров:**
P₁·v₁ = P₂·v₂

**Изменение внутренней энергии:** Δu = 0

**Изменение энтальпии:** Δh = 0

**Работа и теплота [citation:4]:**
q = l = R·T·ln(v₂/v₁) = R·T·ln(P₁/P₂)

**Изменение энтропии:**
Δs = R·ln(v₂/v₁) = R·ln(P₁/P₂)

### 5.4 АДИАБАТНЫЙ ПРОЦЕСС (δq = 0)

**Уравнение процесса (уравнение Пуассона):**
P·vᵏ = const
T·vᵏ⁻¹ = const
T/P⁽ᵏ⁻¹⁾/ᵏ = const

где k = cₚ/cᵥ - показатель адиабаты

**Соотношения параметров:**
T₂/T₁ = (v₁/v₂)ᵏ⁻¹ = (P₂/P₁)⁽ᵏ⁻¹⁾/ᵏ

**Работа:**
l = -Δu = cᵥ·(T₁ - T₂) = (R/(k-1))·(T₁ - T₂) = (1/(k-1))·(P₁v₁ - P₂v₂)

**Теплота:** q = 0

**Изменение энтропии:** Δs = 0

### 5.5 ПОЛИТРОПНЫЙ ПРОЦЕСС

**Уравнение процесса:** P·vⁿ = const, где n - показатель политропы

**Соотношения параметров:**
T₂/T₁ = (v₁/v₂)ⁿ⁻¹ = (P₂/P₁)⁽ⁿ⁻¹⁾/ⁿ

**Работа:**
l = (R/(n-1))·(T₁ - T₂) = (1/(n-1))·(P₁v₁ - P₂v₂)

**Теплоемкость политропного процесса:**
cₙ = cᵥ·(n - k)/(n - 1)

**Теплота:**
q = cₙ·(T₂ - T₁)

**Изменение внутренней энергии:**
Δu = cᵥ·(T₂ - T₁)

**Изменение энтропии:**
Δs = cₙ·ln(T₂/T₁)

### Частные случаи политропного процесса [citation:6]:
- n = 0: P = const (изобарный)
- n = 1: T = const (изотермический)
- n = k: δq = 0 (адиабатный)
- n = ±∞: v = const (изохорный)

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 5.1 (Изотермический процесс)
1 кг воздуха при температуре 27°С и давлении 0.1 МПа изотермически сжимается до давления 0.5 МПа. Определить конечный объем, работу сжатия и количество отведенной теплоты.

**Дано:** m = 1 кг, t = 27°C → T = 300 K, P₁ = 0.1 МПа, P₂ = 0.5 МПа, R(воздуха) = 287 Дж/(кг·К)

**Решение:**
1. Начальный объем:
v₁ = R·T/P₁ = 287·300/(0.1·10⁶) = 86100/10⁵ = 0.861 м³/кг

2. Конечный объем (T=const → P₁v₁ = P₂v₂):
v₂ = v₁·P₁/P₂ = 0.861·0.1/0.5 = 0.861·0.2 = 0.1722 м³/кг

3. Работа сжатия:
l = R·T·ln(P₁/P₂) = 287·300·ln(0.1/0.5) = 86100·ln(0.2) = 86100·(-1.609) = -138.5 кДж/кг
(знак минус означает, что работа совершается над газом)

4. Теплота (для изотермического процесса q = l):
q = -138.5 кДж/кг (теплота отводится)

**Ответ:** v₂ = 0.1722 м³/кг, l = -138.5 кДж/кг, q = -138.5 кДж/кг

### Пример 5.2 (Адиабатный процесс)
Воздух при давлении 0.1 МПа и температуре 20°С адиабатно сжимается до давления 1 МПа. Определить конечные параметры и работу сжатия. k = 1.4.

**Дано:** P₁ = 0.1 МПа, t₁ = 20°C → T₁ = 293 K, P₂ = 1 МПа, k = 1.4

**Решение:**
1. Конечная температура (уравнение адиабаты):
T₂ = T₁·(P₂/P₁)⁽ᵏ⁻¹⁾/ᵏ = 293·(1/0.1)⁰·⁴/¹·⁴ = 293·10⁰·²⁸⁵⁷
10⁰·²⁸⁵⁷ = 10⁰·²⁸⁵⁷ = 1.93
T₂ = 293·1.93 = 565.5 K = 292.5°C

2. Конечный объем:
v₁ = R·T₁/P₁ = 287·293/(0.1·10⁶) = 84191/10⁵ = 0.842 м³/кг
v₂ = v₁·(P₁/P₂)¹/ᵏ = 0.842·(0.1)¹/¹·⁴ = 0.842·0.1⁰·⁷¹⁴³
0.1⁰·⁷¹⁴³ = 0.1⁰·⁷¹⁴³ = 0.166
v₂ = 0.842·0.166 = 0.14 м³/кг

3. Работа сжатия:
l = cᵥ·(T₁ - T₂) = 718·(293 - 565.5) = 718·(-272.5) = -195.7 кДж/кг

**Ответ:** T₂ = 565.5 К, v₂ = 0.14 м³/кг, l = -195.7 кДж/кг

# РАЗДЕЛ 6: ВТОРОЙ ЗАКОН ТЕРМОДИНАМИКИ И ЭНТРОПИЯ
===============================================================================

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### Формулировки второго закона термодинамики [citation:3]:

1. **Клаузиус:** Теплота не может самопроизвольно переходить от холодного тела к горячему.

2. **Томсон (Кельвин):** Невозможно построить периодически действующую машину, единственным результатом которой было бы превращение теплоты в работу.

3. **Энтропийная формулировка:** Энтропия изолированной системы не убывает (dS ≥ 0).

### Энтропия [citation:4]:

**Определение:**
dS = δQ/T [Дж/К]

Энтропия - функция состояния, характеризующая меру неупорядоченности системы. В обратимых процессах изменение энтропии равно приведенной теплоте. В необратимых процессах энтропия возрастает.

**Изменение энтропии в процессах идеального газа:**
Δs = cᵥ·ln(T₂/T₁) + R·ln(v₂/v₁)
Δs = cₚ·ln(T₂/T₁) - R·ln(P₂/P₁)
Δs = cₚ·ln(v₂/v₁) + cᵥ·ln(P₂/P₁)

**Статистический смысл энтропии (Больцман):**
S = k·lnW

где k - постоянная Больцмана, W - термодинамическая вероятность (число микросостояний).

### Цикл Карно [citation:3]:

Идеальный цикл, состоящий из двух изотерм и двух адиабат. Его термический КПД максимален для заданных температур нагревателя T₁ и холодильника T₂:

η_max = 1 - T₂/T₁

Холодильный коэффициент обратного цикла Карно:
ε = T₂/(T₁ - T₂)

### Неравенство Клаузиуса:
∮ δQ/T ≤ 0

Для обратимых циклов: ∮ δQ/T = 0
Для необратимых циклов: ∮ δQ/T < 0

### Эксергия [citation:3]:
Максимальная работа, которую может совершить система при обратимом переходе в равновесие с окружающей средой:

E = (H - H₀) - T₀(S - S₀)

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 6.1
Определить изменение энтропии 1 кг воздуха при нагревании от 100°С до 300°С при постоянном давлении. cₚ = 1.005 кДж/(кг·К).

**Дано:** m = 1 кг, t₁ = 100°C → T₁ = 373 K, t₂ = 300°C → T₂ = 573 K, P = const

**Решение:**
Δs = cₚ·ln(T₂/T₁) = 1.005·ln(573/373) = 1.005·ln(1.536) = 1.005·0.429 = 0.431 кДж/(кг·К)

**Ответ:** Δs = 0.431 кДж/(кг·К)

### Пример 6.2
Тепловая машина работает по циклу Карно с температурой нагревателя 500 К и холодильника 300 К. Определить термический КПД цикла и количество теплоты, отдаваемое холодильнику, если подведенная теплота составляет 100 кДж.

**Дано:** T₁ = 500 K, T₂ = 300 K, Q₁ = 100 кДж

**Решение:**
1. КПД цикла Карно:
η = 1 - T₂/T₁ = 1 - 300/500 = 1 - 0.6 = 0.4 = 40%

2. Работа цикла:
L = η·Q₁ = 0.4·100 = 40 кДж

3. Теплота, отдаваемая холодильнику:
Q₂ = Q₁ - L = 100 - 40 = 60 кДж

**Ответ:** η = 40%, Q₂ = 60 кДж

# РАЗДЕЛ 7: ВОДЯНОЙ ПАР

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### Фазовые состояния воды и водяного пара:

1. **Вода** - жидкость
2. **Влажный насыщенный пар** - смесь кипящей воды и сухого насыщенного пара
3. **Сухой насыщенный пар** - пар при температуре кипения, не содержащий капель жидкости
4. **Перегретый пар** - пар, температура которого выше температуры насыщения при данном давлении

### Основные параметры:

**Степень сухости пара (х)** - массовая доля сухого насыщенного пара во влажном паре:
x = m_сух.пара / (m_воды + m_сух.пара)

Для кипящей воды: x = 0
Для сухого насыщенного пара: x = 1

**Параметры влажного пара:**
v_x = v' + x·(v'' - v') = v'·(1-x) + v''·x
h_x = h' + x·(h'' - h') = h' + x·r
s_x = s' + x·(s'' - s')

где:
v', h', s' - параметры кипящей воды
v'', h'', s'' - параметры сухого насыщенного пара
r = h'' - h' - удельная теплота парообразования

### Уравнение Клапейрона-Клаузиуса [citation:6]:
dP/dT = r / (T·(v'' - v'))

### Таблицы водяного пара:
Для практических расчетов используют таблицы термодинамических свойств воды и водяного пара, а также hs-диаграмму [citation:3].

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 7.1
Определить параметры влажного насыщенного пара при давлении P = 1 МПа и степени сухости x = 0.9.

**Дано:** P = 1 МПа, x = 0.9

**Решение:**
По таблицам насыщенного водяного пара при P = 1 МПа (t_s = 179.9°C):
v' = 0.001127 м³/кг, v'' = 0.1946 м³/кг
h' = 762.7 кДж/кг, h'' = 2778 кДж/кг, r = 2015.3 кДж/кг
s' = 2.138 кДж/(кг·К), s'' = 6.586 кДж/(кг·К)

1. Удельный объем:
v_x = v'·(1-x) + v''·x = 0.001127·0.1 + 0.1946·0.9 = 0.0001127 + 0.17514 = 0.17525 м³/кг

2. Энтальпия:
h_x = h' + x·r = 762.7 + 0.9·2015.3 = 762.7 + 1813.77 = 2576.47 кДж/кг

3. Энтропия:
s_x = s' + x·(s'' - s') = 2.138 + 0.9·(6.586 - 2.138) = 2.138 + 0.9·4.448 = 2.138 + 4.003 = 6.141 кДж/(кг·К)

**Ответ:** v_x = 0.175 м³/кг, h_x = 2576.5 кДж/кг, s_x = 6.141 кДж/(кг·К)

### Пример 7.2
Определить количество теплоты, необходимое для получения 5 кг перегретого пара при давлении P = 2 МПа и температуре t = 400°С из воды при 0°С.

**Дано:** m = 5 кг, P = 2 МПа, t = 400°C, t_нач = 0°C

**Решение:**
По таблицам перегретого пара при P = 2 МПа, t = 400°C:
h = 3248 кДж/кг

Энтальпия воды при 0°C: h_0 ≈ 0 кДж/кг

Количество теплоты на получение 1 кг пара:
q = h - h_0 = 3248 кДж/кг

Общее количество теплоты:
Q = m·q = 5·3248 = 16240 кДж = 16.24 МДж

**Ответ:** Q = 16.24 МДж

# РАЗДЕЛ 8: ПРОЦЕССЫ ИЗМЕНЕНИЯ СОСТОЯНИЯ ВОДЯНОГО ПАРА
===============================================================================

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

Для расчета процессов изменения состояния водяного пара используют таблицы термодинамических свойств и hs-диаграмму [citation:3].

### 8.1 ИЗОХОРНЫЙ ПРОЦЕСС (v = const)

Конечные параметры определяют по таблицам, используя условие v₁ = v₂.

Количество теплоты:
q = u₂ - u₁ = (h₂ - P₂v₂) - (h₁ - P₁v₁)

### 8.2 ИЗОБАРНЫЙ ПРОЦЕСС (P = const)

Теплота процесса равна изменению энтальпии:
q = h₂ - h₁

Работа:
l = P·(v₂ - v₁)

### 8.3 ИЗОТЕРМИЧЕСКИЙ ПРОЦЕСС (T = const)

Конечные параметры определяют по температуре и одному из параметров.

Теплота процесса:
q = T·(s₂ - s₁)

Изменение внутренней энергии:
Δu = u₂ - u₁

Работа:
l = q - Δu

### 8.4 АДИАБАТНЫЙ ПРОЦЕСС (s = const)

Для обратимого адиабатного процесса s₂ = s₁.

Работа:
l = -Δu = u₁ - u₂

В hs-диаграмме адиабатный процесс изображается вертикальной прямой (s = const).

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 8.1 (Изобарный процесс)
1 кг водяного пара при давлении P = 1 МПа и степени сухости x₁ = 0.9 изобарно нагревается до температуры t₂ = 300°С. Определить количество подведенной теплоты, изменение внутренней энергии и совершенную работу.

**Дано:** m = 1 кг, P = 1 МПа, x₁ = 0.9, t₂ = 300°C

**Решение:**
1. Параметры начального состояния (влажный пар):
По таблицам при P = 1 МПа:
t_s = 179.9°C, h' = 762.7 кДж/кг, h'' = 2778 кДж/кг, r = 2015.3 кДж/кг
v' = 0.001127 м³/кг, v'' = 0.1946 м³/кг

h₁ = h' + x₁·r = 762.7 + 0.9·2015.3 = 762.7 + 1813.77 = 2576.47 кДж/кг
v₁ = v' + x₁·(v'' - v') = 0.001127 + 0.9·(0.1946 - 0.001127) = 0.001127 + 0.9·0.193473 = 0.001127 + 0.174126 = 0.175253 м³/кг
u₁ = h₁ - P·v₁·10³ = 2576.47 - 1·10⁶·0.175253·10⁻³ = 2576.47 - 175.253 = 2401.22 кДж/кг

2. Параметры конечного состояния (перегретый пар):
По таблицам перегретого пара при P = 1 МПа, t₂ = 300°C:
h₂ = 3051 кДж/кг, v₂ = 0.2579 м³/кг
u₂ = h₂ - P·v₂·10³ = 3051 - 1·10⁶·0.2579·10⁻³ = 3051 - 257.9 = 2793.1 кДж/кг

3. Теплота процесса:
q = h₂ - h₁ = 3051 - 2576.47 = 474.53 кДж/кг

4. Работа:
l = P·(v₂ - v₁)·10³ = 1·10⁶·(0.2579 - 0.175253)·10⁻³ = 10³·0.082647 = 82.65 кДж/кг

5. Изменение внутренней энергии:
Δu = u₂ - u₁ = 2793.1 - 2401.22 = 391.88 кДж/кг

Проверка по первому закону: q = Δu + l = 391.88 + 82.65 = 474.53 кДж/кг

**Ответ:** q = 474.53 кДж/кг, Δu = 391.88 кДж/кг, l = 82.65 кДж/кг

Водяной пар при давлении P₁ = 1.5 МПа и температуре t₁ = 300°С адиабатно расширяется до давления P₂ = 0.1 МПа. Определить конечные параметры пара и работу расширения, используя hs-диаграмму.

# РАЗДЕЛ 9: ВЛАЖНЫЙ ВОЗДУХ

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

**Влажный воздух** - смесь сухого воздуха и водяного пара.

### Основные характеристики [citation:3]:

1. **Абсолютная влажность** - масса водяного пара в 1 м³ влажного воздуха (численно равна плотности пара ρ_п).

2. **Относительная влажность (φ)** - отношение парциального давления пара к давлению насыщения при данной температуре:
φ = p_п / p_н = ρ_п / ρ_н

3. **Влагосодержание (d)** - масса пара, приходящаяся на 1 кг сухого воздуха [г/кг с.в.]:
d = 622·(p_п / (P - p_п)) [г/кг с.в.]
где P - общее давление влажного воздуха

4. **Температура точки росы (t_р)** - температура, до которой нужно охладить влажный воздух, чтобы он стал насыщенным (φ = 100%) при постоянном влагосодержании.

### Энтальпия влажного воздуха:
h = h_с.в. + (d/1000)·h_п [кДж/кг с.в.]
где h_с.в. = c_с.в.·t = 1.005·t - энтальпия сухого воздуха
h_п = 2500 + 1.93·t - энтальпия пара (приближенно)

### hd-диаграмма влажного воздуха [citation:3]:
Для практических расчетов процессов с влажным воздухом используют hd-диаграмму (построена для определенного барометрического давления, обычно P = 745 мм рт.ст.).

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 9.1
Определить относительную влажность, влагосодержание и энтальпию влажного воздуха при температуре t = 25°С и парциальном давлении пара p_п = 1.5 кПа. Барометрическое давление P = 100 кПа.

**Дано:** t = 25°C, p_п = 1.5 кПа, P = 100 кПа

**Решение:**
1. Давление насыщения при t = 25°C (по таблицам водяного пара):
p_н = 3.17 кПа

2. Относительная влажность:
φ = p_п / p_н = 1.5 / 3.17 = 0.473 = 47.3%

3. Влагосодержание:
d = 622·(p_п / (P - p_п)) = 622·(1.5 / (100 - 1.5)) = 622·(1.5/98.5) = 622·0.01523 = 9.47 г/кг с.в.

4. Энтальпия:
h_п = 2500 + 1.93·t = 2500 + 1.93·25 = 2500 + 48.25 = 2548.25 кДж/кг
h = 1.005·t + (d/1000)·h_п = 1.005·25 + (9.47/1000)·2548.25 = 25.125 + 0.00947·2548.25 = 25.125 + 24.13 = 49.255 кДж/кг с.в.

**Ответ:** φ = 47.3%, d = 9.47 г/кг с.в., h = 49.26 кДж/кг с.в.

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### Истечение газов и паров [citation:3]:

При истечении через сопла потенциальная энергия давления преобразуется в кинетическую энергию потока.

**Скорость истечения из сопла:**
Для идеального газа при адиабатном истечении:
c = √(2·(h₁ - h₂)) = √(2·cₚ·(T₁ - T₂))

Для идеального газа с постоянной теплоемкостью:
c = √(2·(k/(k-1))·R·T₁·(1 - (P₂/P₁)⁽ᵏ⁻¹⁾/ᵏ))

**Массовый расход:**
G = f·c / v₂ = f·√(2·(h₁ - h₂)) / v₂

где f - площадь выходного сечения сопла

### Критическое давление и скорость:

При истечении через суживающееся сопло максимальная скорость достигается, когда давление в выходном сечении становится равным критическому:

**Критическое отношение давлений:**
β_кр = P_кр / P₁ = (2/(k+1))ᵏ/⁽ᵏ⁻¹⁾

Для двухатомных газов (k=1.4): β_кр = 0.528
Для перегретого пара (k=1.3): β_кр = 0.546

**Критическая скорость:**
c_кр = √(2·(k/(k+1))·R·T₁) = √(k·P_кр·v_кр)

### Сопло Лаваля:
Для получения сверхзвуковых скоростей (при P₂/P₁ < β_кр) используют расширяющиеся сопла (сопла Лаваля).

### Дросселирование [citation:3]:

Процесс понижения давления при прохождении газа через местное сопротивление (вентиль, заслонку). При дросселировании энтальпия остается постоянной: h₂ = h₁.

Для идеального газа температура при дросселировании не изменяется. Для реальных газов и паров температура может изменяться (эффект Джоуля-Томсона).

**Температура инверсии** - температура, при которой эффект Джоуля-Томсона меняет знак.

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 10.1
Воздух при давлении P₁ = 1 МПа и температуре t₁ = 27°С истекает через суживающееся сопло в среду с давлением P₂ = 0.5 МПа. Определить скорость истечения и массовый расход, если площадь выходного сечения сопла f = 50 мм². Показатель адиабаты k = 1.4, R = 287 Дж/(кг·К).

**Дано:** P₁ = 1 МПа = 10⁶ Па, t₁ = 27°C → T₁ = 300 K, P₂ = 0.5 МПа = 5·10⁵ Па, f = 50 мм² = 5·10⁻⁵ м²

**Решение:**
1. Определим критическое отношение давлений:
β_кр = (2/(k+1))ᵏ/⁽ᵏ⁻¹⁾ = (2/2.4)¹·⁴/⁰·⁴ = (0.8333)³·⁵ = 0.528

2. Фактическое отношение давлений:
β = P₂/P₁ = 0.5/1 = 0.5

Так как β = 0.5 < β_кр = 0.528, режим истечения сверхкритический. Давление в выходном сечении равно критическому: P_кр = β_кр·P₁ = 0.528·1 = 0.528 МПа.

3. Критическая температура:
T_кр = T₁·(P_кр/P₁)⁽ᵏ⁻¹⁾/ᵏ = 300·(0.528)⁰·⁴/¹·⁴ = 300·(0.528)⁰·²⁸⁵⁷
(0.528)⁰·²⁸⁵⁷ = 0.528⁰·²⁸⁵⁷ = 0.825
T_кр = 300·0.825 = 247.5 K

4. Критическая скорость:
c_кр = √(2·(k/(k+1))·R·T₁) = √(2·(1.4/2.4)·287·300) = √(2·0.5833·86100) = √(2·0.5833·86100) = √(100400) = 317 м/с

5. Удельный объем в критическом сечении:
v_кр = R·T_кр/P_кр = 287·247.5/(0.528·10⁶) = 71032.5/(5.28·10⁵) = 0.1345 м³/кг

6. Массовый расход:
G = f·c_кр / v_кр = 5·10⁻⁵·317 / 0.1345 = 0.01585 / 0.1345 = 0.118 кг/с

**Ответ:** c_кр = 317 м/с, G = 0.118 кг/с

# РАЗДЕЛ 11: ЦИКЛЫ ТЕПЛОВЫХ ЭНЕРГЕТИЧЕСКИХ УСТАНОВОК

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### 11.1 ЦИКЛ РЕНКИНА [citation:3]

Цикл Ренкина - основной цикл паротурбинных установок (ПТУ). Состоит из процессов:
1-2: Адиабатное расширение пара в турбине
2-3: Конденсация отработавшего пара в конденсаторе
3-4: Сжатие воды в насосе
4-1: Подвод теплоты в котле (изобарный процесс)

**Термический КПД цикла Ренкина:**
η_t = (h₁ - h₂) / (h₁ - h₃)

**Удельный расход пара:**
d = 3600 / (h₁ - h₂) [кг/(кВт·ч)]

**Удельный расход теплоты:**
q = 3600 / η_t [кДж/(кВт·ч)]

### 11.2 ЦИКЛ С ПРОМЕЖУТОЧНЫМ ПЕРЕГРЕВОМ ПАРА

Для повышения КПД и уменьшения влажности пара в последних ступенях турбины применяют промежуточный (вторичный) перегрев пара.

### 11.3 РЕГЕНЕРАТИВНЫЙ ЦИКЛ

Подогрев питательной воды паром, отбираемым из промежуточных ступеней турбины, повышает термический КПД установки.

### 11.4 ГАЗОТУРБИННЫЕ УСТАНОВКИ (ГТУ) [citation:6]

Простой цикл ГТУ с подводом тепла при постоянном давлении (цикл Брайтона):
1-2: Адиабатное сжатие в компрессоре
2-3: Изобарный подвод тепла в камере сгорания
3-4: Адиабатное расширение в турбине
4-1: Изобарный отвод тепла в атмосферу

**Термический КПД цикла ГТУ:**
η_t = 1 - 1/π⁽ᵏ⁻¹⁾/ᵏ
где π = P₂/P₁ - степень повышения давления в компрессоре

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 11.1 (Цикл Ренкина)
Определить термический КПД паротурбинной установки, работающей по циклу Ренкина, если параметры пара перед турбиной: P₁ = 9 МПа, t₁ = 500°С. Давление в конденсаторе P₂ = 4 кПа.

**Дано:** P₁ = 9 МПа = 90 бар, t₁ = 500°C, P₂ = 4 кПа = 0.04 бар

**Решение:**
Используем hs-диаграмму или таблицы водяного пара.

1. Точка 1 (перегретый пар):
По таблицам при P₁ = 9 МПа, t₁ = 500°C:
h₁ = 3380 кДж/кг, s₁ = 6.66 кДж/(кг·К)

2. Точка 2 (влажный пар после расширения в турбине):
Процесс 1-2 адиабатный (s₂ = s₁ = 6.66 кДж/(кг·К))
При P₂ = 4 кПа по таблицам насыщения:
t_s = 29°C, h' = 121.4 кДж/кг, h'' = 2554 кДж/кг, s' = 0.422 кДж/(кг·К), s'' = 8.47 кДж/(кг·К)

Степень сухости:
x₂ = (s₂ - s')/(s'' - s') = (6.66 - 0.422)/(8.47 - 0.422) = 6.238/8.048 = 0.775

h₂ = h' + x₂·(h'' - h') = 121.4 + 0.775·(2554 - 121.4) = 121.4 + 0.775·2432.6 = 121.4 + 1885.3 = 2006.7 кДж/кг

3. Точка 3 (конденсат на выходе из конденсатора):
h₃ = h' = 121.4 кДж/кг

4. Термический КПД:
η_t = (h₁ - h₂)/(h₁ - h₃) = (3380 - 2006.7)/(3380 - 121.4) = 1373.3/3258.6 = 0.421 = 42.1%

5. Удельный расход пара:
d = 3600/(h₁ - h₂) = 3600/1373.3 = 2.62 кг/(кВт·ч)

**Ответ:** η_t = 42.1%, d = 2.62 кг/(кВт·ч)

# РАЗДЕЛ 12: ЦИКЛЫ ХОЛОДИЛЬНЫХ УСТАНОВОК И ТЕПЛОВЫХ НАСОСОВ

## ТЕОРЕТИЧЕСКИЕ ОСНОВЫ

### Обратные циклы [citation:3]:

В холодильных установках реализуются обратные циклы, на осуществление которых затрачивается работа извне.

**Холодильный коэффициент:**
ε = Q₂ / L = Q₂ / (Q₁ - Q₂)

где Q₂ - теплота, отведенная от охлаждаемого тела, L - затраченная работа.

### 12.1 ЦИКЛ ВОЗДУШНОЙ ХОЛОДИЛЬНОЙ УСТАНОВКИ

Состоит из процессов:
1-2: Адиабатное сжатие в компрессоре
2-3: Изобарное охлаждение в теплообменнике
3-4: Адиабатное расширение в детандере
4-1: Изобарный нагрев в охлаждаемом объеме

Холодильный коэффициент цикла:
ε = (T₁ - T₄) / ((T₂ - T₁) - (T₃ - T₄))

### 12.2 ЦИКЛ ПАРОВОЙ КОМПРЕССИОННОЙ ХОЛОДИЛЬНОЙ УСТАНОВКИ

Наиболее распространенный тип холодильных машин. Рабочее тело - хладагент (аммиак, фреоны и др.).

Холодильный коэффициент:
ε = q₂ / l = (h₁ - h₄) / (h₂ - h₁)

### 12.3 ТЕПЛОВЫЕ НАСОСЫ

Устройства для передачи теплоты от холодного источника к горячему за счет затраты работы.

**Отопительный коэффициент:**
μ = Q₁ / L = (Q₂ + L) / L = 1 + ε

## ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 12.1
Парокомпрессионная холодильная установка работает на хладагенте R12. Температура кипения t₀ = -15°С, температура конденсации t_к = 25°С. Определить холодильный коэффициент и холодопроизводительность установки, если массовый расход хладагента G = 0.1 кг/с.

**Дано:** t₀ = -15°C, t_к = 25°C, G = 0.1 кг/с

**Решение:**
По таблицам для R12 при t₀ = -15°C:
h₁ = 348 кДж/кг (пар на входе в компрессор)
При t_к = 25°C:
h₃ = h₄ = 230 кДж/кг (жидкость после конденсатора)

Энтальпия пара после сжатия (по hs-диаграмме при s₂ = s₁ и P₂ = P_к):
h₂ = 375 кДж/кг

1. Удельная холодопроизводительность:
q₂ = h₁ - h₄ = 348 - 230 = 118 кДж/кг

2. Удельная работа компрессора:
l = h₂ - h₁ = 375 - 348 = 27 кДж/кг

3. Холодильный коэффициент:
ε = q₂ / l = 118 / 27 = 4.37

4. Полная холодопроизводительность:
Q₂ = G·q₂ = 0.1·118 = 11.8 кВт

**Ответ:** ε = 4.37, Q₂ = 11.8 кВт
