# Команда ИИ-агентов: локальный преподаватель по термодинамике

Этот проект реализует локального ИИ-преподавателя по технической термодинамике. Он отвечает студентам на основе учебных материалов, используя:
- 📚 Учебные материалы в формате PDF (папка /books)
- 🔍 Векторный поиск для нахождения релевантных фрагментов
- 🤖 Локальная LLM через Ollama
- 💬 Историю диалога для поддержания контекста

Работа осуществляется полностью локально: без интернета, без API-ключей и без ограничений по количеству запросов.

---

## ✨ Возможности

- 📖 Работа с материалами
  - Автоматическая загрузка и индексация PDF-файлов из папки /books
  - Разбиение документов на чанки с сохранением контекста
  - Векторное хранение для семантического поиска

- 🤖 Локальный ИИ
  - Поддержка любой модели Ollama (qwen3, llama3, mistral, gemma)
  - Полностью офлайн-работа
  - Неограниченное число запросов

- 💬 Умный диалог
  - Понимание контекста диалога
  - Ответы на основе учебных материалов
  - Честное «не знаю», если информации нет

- 🛡️ Безопасность
  - Фильтрация политических тем
  - Защита от токсичных выражений
  - Ограничение частоты запросов для предотвращения спама

## 🔧 Установка

1. Клонируйте репозиторий и перейдите в него
   - `git clone https://github.com/AI-agent-team-3/dkarabarin
   -  cd Команда-ИИ-агентов-3`

2. Установите Ollama и модель
   - Установите Ollama с официального сайта: https://ollama.com
   - Скачать модель (рекомендуется qwen3:4b для русского языка):
     - `ollama pull qwen3:4b`

   Альтернативы:
   - `ollama pull llama3.2:3b`  (английский, 3B)
   - `ollama pull mistral:7b`
   - `ollama pull gemma2:2b`

3. Создайте виртуальное окружение и установите зависимости
   - Windows:
     - `python -m venv .venv`
     - `.venv\Scripts\activate`
   - Linux/macOS:
     - `python3 -m venv .venv`
     - `source .venv/bin/activate`
   - `pip install -r requirements.txt`

4. Подготовьте учебные материалы
   - Создайте папку для книг:
     - `mkdir books`
   - Поместите PDF-файлы в папку books/
     - Примеры: `books/termodinamika.pdf`, `books/fizika.pdf`

5. Запуск бота
   - `python bot-local-ollama.py`

## 📚 Использование

Первый запуск выполняет автоматическую подготовку:
- Проверка наличия Ollama
- Индексацию PDF-файлов из /books
- Создание векторной базы данных в /chroma_db
- Запуск интерактивного интерфейса

Пример рабочей сессии:
- Вы: объясните первый закон термодинамики
- Робот: ответ на основе ваших учебников
- Вы: а формула?
- Робот: ответ с учётом контекста

## Команды и статистика

Команды доступны через CLI:
- `/help` — справка
- `/clear` — очистить экран
- `/stats` — статистика базы знаний
- `/history` — история диалога
- `/clear_history` — очистить историю
- `/models` — список доступных моделей Ollama
- `/switch` — переключить текущую модель
- `/exit` — выйти

Пример вывода статистики базы знаний:
- Всего чанков: 45
- Коллекция: knowledge_base
- Источники:
  - termodinamika.pdf: 30 чанков
  - fizika.pdf: 15 чанков

## Конфигурация

- `DEFAULT_MODEL` в `bot-local-ollama.py` управляет моделью по умолчанию:
  - `DEFAULT_MODEL = "qwen3:4b"`  — для русского языка
  - Варианты: `llama3.2:3b`, `mistral:7b`, `gemma2:2b`

Примечания:
- Убедитесь, что Ollama запущена перед использованием
- Для больших PDF-файлов требуется достаточная оперативная память
- Первая индексация может занять время
- Все данные хранатся локально

---

## Решение проблем

- Ollama не найдена — установите Ollama и запустите его
- Нет ответа от модели — проверьте, что модель скачана (ollama list)
- Ошибка загрузки PDF — убедитесь в целостности файлов и расширении .pdf
- Медленная работа — попробуйте легкую модель (gemma2:2b)
- Неправильные ответы — улучшите качество PDF и добавьте материалов
