# 🔥 ИИ преподаватель по ТТД и ТМО

**Локальный ИИ-ассистент** для изучения технической термодинамики (ТТД) и тепломассообмена (ТМО) с поддержкой RAG (Retrieval-Augmented Generation), веб-поиском (Tavily/DuckDuckGo), **многоуровневой системой безопасности** и современным веб-интерфейсом с поддержкой формул.

---

## 📚 Особенности

### 🤖 Основные функции
- **RAG на PDF-документах** — загружайте учебники и методички в папку `books/`
- **Локальная LLM через Ollama** — бесплатно, без интернета, полный контроль над данными
- **Веб-поиск с автовыбором** — Tavily (есть ключ) или DuckDuckGo (бесплатно, без ключа)
- **Гибридный поиск** — приоритет: PDF → веб-поиск → LLM
- **Полная наблюдаемость через Langfuse** — трассировка всех запросов (опционально)
- **Система Guardrails** — система безопасности
### 🛡️ Система безопасности (Guardrails)
- **Защита от Prompt Injection** — блокировка попыток взлома и изменения поведения
- **Защита от Jailbreak атак** — блокировка DAN, AIM и других техник обхода
- **Фильтрация персональных данных (PII)** — автоматическое удаление email, телефонов, паспортных данных, СНИЛС
- **Академическая честность** — отклонение запросов на списывание и готовые ответы
- **Rate Limiting** — ограничение частоты запросов (10 запросов в минуту)
- **Логирование атак** — все инциденты сохраняются в `security_log.json`
- **CVE-style оценка уязвимостей** — критичность каждой угрозы

### 🌐 Веб-интерфейс
- 💬 **Чат с историей сообщений** — сохранение в localStorage
- 📐 **Поддержка LaTeX формул** — красивое отображение через MathJax
- 📊 **Панель статуса** — отображение состояния RAG, поиска и модели
- 🗑️ **Кнопка очистки истории**
- 📱 **Адаптивный дизайн** — работает на любом устройстве
- 🔒 **Встроенная безопасность** — все проверки на стороне сервера

### 🧪 Тестирование безопасности
- **Расширенная оценка с CVE-style severity scoring**
- **15+ тестовых сценариев** (Prompt Injection, Jailbreak, PII, Academic Integrity)
- **Автоматический отчет** в JSON с метриками безопасности
- **Security Score** — общая оценка от 0 до 100

## 🚀 Быстрый старт
1. **Установка Python 3.11+**
2. **Установка Ollama**
    - `ollama pull qwen3:4b`
3. **Получение API ключей**
    - Tavily API (веб-поиск): зарегистрируйтесь на [app.tavily.com](https://app.tavily.com), получите бесплатный API ключ (1000 запросов/мес)
    - Langfuse (опционально): запустите локально через Docker или используйте [cloud.langfuse.com](https://cloud.langfuse.com)
4. **Клонирование и установка**
    ```bash
    git clone <repository-url>
    cd thermodynamics-rag
    python -m venv venv
    ```
    - Активация виртуальной среды:
        - Windows:`venv\Scripts\activate.bat`
        - Linux/Mac:`source venv/bin/activate`
    - Установка зависимостей: `pip install -r requirements.txt`
5. **Настройка окружения**
    ```bash
    cp .env.example .env
    ```
    - Отредактируйте `.env` и вставьте ваши ключи API
6. **Подготовка базы знаний**
    ```bash
    mkdir books
    # Поместите PDF-файлы по термодинамике в папку books/
    ```
7. **Запуск Langfuse (опционально)**
    ```bash
    docker compose up -d
    ```
8. **Запуск ассистента**
    - Консольная версия:
        ```bash
        python bot-local.py
        ```
    - Веб-версия (новая!):
        ```bash
        python -m uvicorn web.api:app --reload --host 0.0.0.0 --port 8000
        ```
        Затем откройте в браузере: [http://localhost:8000](http://localhost:8000)
     - Тест безопасности, в новом окне терминала:
       ```bash
        python security_tester.py

# 🐛 Советы по устранению проблем
## Ollama не запускается
    ollama list
    ollama serve
    ollama pull qwen3:4b
## Langfuse не подключается
    docker ps
    docker compose up -d
    curl http://localhost:3000/api/health
## Нет PDF-файлов
    mkdir books
### добавьте свои PDF в папку books/
## Проблемы с памятью
    Уменьшите параметры CHUNK_SIZE, CHUNK_OVERLAP, K_RETRIEVAL в файле rag.py.
## Веб-интерфейс не открывается
### Проверьте, что сервер запущен
    curl http://localhost:8000/api/health
### Проверьте, занят ли порт
    netstat -ano | findstr :8000
### Перезапустите сервер
    python -m uvicorn web.api:app --reload --port 8000
## Формулы не отображаются
    Очистите кэш браузера (Ctrl+Shift+Delete).
    Проверьте консоль браузера (F12) на ошибки MathJax.
    Обновите страницу (Ctrl+F5).
